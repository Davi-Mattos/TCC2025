# INSTALAÇÃO

1. Requisitos do Sistema

Navegador atualizado (Chrome, Firefox, etc.).

Servidor web : Apache, Nginx ou Live Server.

Visual Studio Code

Bibliotecas Node.js, bcrypt, jsonwebtoken, mysql2, multer, express-session, cookie-parser e Express.

NPM pra instalação das bibliotecas:

`npm install express cookie-parser express-session multer mysql2 bcrypt jsonwebtoken`

Banco de dados MySQL necessário 

2. Preparação do Ambiente

Explique como preparar o ambiente:

Instalar o Node.js 

Instalar o XAMPP e dar start na planilha do "mySQL"

Criar banco de dados com o nome tcc2025 e executar com o codigo escrito aqui: 

...\Tcc2025\docs\banco.sql

3. Execução

Abrir o app.js no Visual Studio Code

No terminal escrever: node app.js

acessar o site no link: `http://localhost:3000``
